# Amazon Clone

Responsive Amazon homepage clone for MLSC Web Development Task.

## Features

- Amazon homepage with carousel
- Login/Signup form with validation
- Dark/Light theme toggle
- Fully responsive design

## Tech Stack

HTML, CSS, JavaScript, Bootstrap 5.3

## Form Validation

**Sign In:**
- Valid email required
- Password min 6 characters

**Sign Up:**
- Name min 2 characters
- Valid email required
- Password min 6 characters
- Passwords must match

## Theme Toggle

Click the button (top-right) to switch themes. Preference is saved automatically.

## Responsive

Works on mobile, tablet, and desktop.

## How to Run

Open `index.html` in any browser.
