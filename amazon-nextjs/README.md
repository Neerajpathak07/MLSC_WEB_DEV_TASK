# Amazon Landing Page Clone

## Overview

Temporary project built with Next.js, made for MLSC Web Dev Task.

## Installation

### Prerequisites
- Node.js
- npm

### Steps
1. Clone the repository:
   ```sh
   git clone https://github.com/AdityaBavadekar/mlsc_web_dev_task_temp
   cd mlsc_web_dev_task_temp/amazon-nextjs
   ```
2. Install dependencies:

- Install pnpm:
   ```sh
   npm install -g pnpm
   ```
- Install project dependencies:
   ```sh
   pnpm install
   ```

## Usage

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
